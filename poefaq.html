FAQ

Why does this page take so long to load?

The PoE trade website has a rate-limiter to keep people from writing scripts that overwhelm the trade site. The rate limiter only allows 1 search per second from each IP address. Webpages being run on a server - like I'm sure poe.ninja is, for example - get around this by caching their search results locally, and only refreshing their searches every so often.

My website is not being hosted on a server, it's being hosted on an AWS bucket. AWS buckets have limited storage space and can't do any calculations on their own. When you load this webpage you're downloading the code and running it from your own computer. Therefore you are limited to 1 search per second. There are 57 itemized prophecy recipies, each being made up of three items (the prophecy, the original item, and the upgraded item). So to load them all you're doing 171 requests (57 x 3). At one per second that's about 3 minutes.
